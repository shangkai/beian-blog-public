ls 查看当前目录下的文件（或文件夹）

pwd print working directory, 打印工作目录，它会显示我们当前所在的目录路径。

cd 目录切换

mkdir 创建文件夹

touch 创建一个空文件

rm 删除文件

vi 向指定文件中插入内容

echo 文字输出

cat 查看文件中的内容

cp 拷贝文件

mv 移动文件

exit 可以直接退出窗口
